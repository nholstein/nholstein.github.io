pub fn sha_en(&self) -> SHA_EN_R[src]

Bit 1 - SHA256 enable. If 0, SHA engine won't initiate compression, this is used to stop operation of the SHA engine until configuration has been done. When the SHA engine is disabled the digest is cleared.
