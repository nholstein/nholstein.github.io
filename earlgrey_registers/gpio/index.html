masked_oe_lower

GPIO write Output Enable lower with mask. Masked write for DATA_OE[15:0], the register that controls output mode for GPIO pins [15:0]. Upper 16 bits of this register are used as mask. Writing lower 16 bits of the register changes DATA_OE[15:0] value if mask bits are set. Read-back of this register returns upper 16 bits as zero and lower 16 bits as DATA_OE[15:0].

masked_oe_upper
